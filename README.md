# Du développement au déploiement d'applications web (Cours)

Ce dépôt regroupe plusieurs implémentations réalisées en cours autour des **architectures API** et des **services web**.

## Objectif

Ce dépôt a une finalité pédagogique et s’inscrit dans un cadre d’apprentissage.

Mettre en pratique :

- La conception d’API REST
- La mise en place d’un service SOAP en Java
- La consommation d’API via un client PHP
- L’utilisation de Flask avec une base MySQL

## Organisation 

Le dépôt est structuré en plusieurs modules indépendants :

- **API REST (Python)**  
    Implémentation simple de gestion des étudiants  
    → Voir le README du dossier
    
- **Service Web SOAP (Java)**  
    Implémentation d’un service SOAP avec modèle de données  
    → Voir le README du dossier
    
- **Client PHP (tests API)**  
    Interface et scripts de test pour consommer l’API Flask  
    → Voir le README du dossier
    
- **API REST Flask + MySQL**  
    API complète avec persistance des données  
    → Voir le README du dossier
    
## Technologies utilisées

- Python (Flask)
- Java (SOAP)
- PHP (XAMPP)
- MySQL

## Remarque

Les détails techniques sont disponibles dans les README spécifiques à chaque module.
